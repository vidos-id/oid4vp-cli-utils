# Release v0.9.0

## Highlights

- **Issuer metadata page**: Added a `/metadata` page that displays OpenID4VCI issuer metadata with the x5c certificate chain formatted as PEM, including copy buttons.
- **Wallet-cli interactive delete actions**: Added interactive delete confirmation actions to wallet-cli.

## New Features

- Issuer web now includes a `/metadata` page showing credential issuer metadata with formatted PEM certificate chains and copy functionality.
- Wallet-cli now supports interactive delete actions for managing stored credentials.

## Bug Fixes

- Corrected `birth_date` field to `birthdate` in demo scripts.
- Fixed wallet-cli interactive flow issues.
- Aligned issuer template `vct` configuration.

## Other Changes

- Refactored issuer web predefined templates for improved maintainability.
- Consolidated page layouts through a shared `PageShell` component to eliminate DOM shape changes between routes.
- Fixed scrollbar-induced layout shift by adding `scrollbar-gutter: stable` on `<html>`.
